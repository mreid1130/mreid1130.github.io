Scuff

Scuff is an npm module and scraper utility tool built on wrappers for cheerio, request, and phantomJS. Implements retries with exponential backoff and error handling to reduce connection errors when scraping sites.

FallSpace

FallSpace is a space shooter/survival game built with HTML, CSS, JavaScript, and jQuery. Built as a part of my Phase 2 project at Dev Bootcamp.

Footballurrito

Footballurrito is a web game built with HTML, CSS, JavaScript, and jQuery. Inspired by the classic web game, Fishy. Built as a part of my Phase 2 project at Dev Bootcamp.

StatCheck

StatCheck is a simple, single page site featuring a custom dropdown and look-ahead search feature, allowing a user to query any NBA player's career statistics. Client built using JavaScript, jQuery, HTML, and CSS and a Node server which supports a daily scraper cron.

Email Notifier

Web scraper and email notification system. Built to notify users when certain data is scraped from a site. API is currently in development and built using Mailgun's API, AWS's SQS, and Node.

Upstarter

Mobile app used to connect job seekers with startups. Ranked startups with a "momentum score" calculated based on fundraising momentum. Built on Ionic and Node. Currently out of order due to LinkedIn's deprecation of it's API.
